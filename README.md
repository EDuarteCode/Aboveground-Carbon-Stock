# 🌳 Aboveground Carbon Modeling

## Supplementary Material  
**Manuscript: Farmers’ actions towards sustainability: An evidence-based methodology for native forest ecosystems supporting low-carbon livestock in grazed dairy systems**

---

## 📌 Overview

This repository provides the **Supplementary Script (S1)** used in the study to model **aboveground carbon stocks** using **Sentinel-2 imagery** and **field plot data** within **Google Earth Engine (GEE)**.

The script is provided as an **example workflow** to illustrate the modeling process used in the study.

Due to **data confidentiality agreements** and **landowner privacy restrictions**, some of the original input datasets cannot be publicly shared. Therefore, this repository includes **representative datasets** to demonstrate the methodology and processing steps.

Access to the original datasets **may be granted upon reasonable request**, subject to data-sharing agreements.

---

## ⚙️ Workflow Description

The script implements the following steps:

- Sentinel-2 image preprocessing  
- Cloud masking and filtering  
- Spectral index calculation  
- Random Forest model training  
- Aboveground carbon stock prediction  
- Map visualization  
- Export of results
